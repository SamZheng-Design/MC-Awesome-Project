## 🔧 API 端点

### 初始化投后数据
```bash
POST /api/init-post-investment?force=true
```

### 获取回款统计
```bash
GET /api/cashflow-stats
```

### 获取单个标的回款详情
```bash
GET /api/deals/:id/cashflows
```

### 获取已投资标的列表
```bash
GET /api/investor/deals
```

### 获取回款记录列表
```bash
GET /api/investor/cashflows
```

## 📝 数据特点

1. **真实模拟**：投资时间分布符合真实投资节奏，早期试点、中期扩张、后期精选
2. **频率多样**：涵盖每日、每周、每月三种回款频率
3. **行业覆盖**：覆盖餐饮、零售、服务、文娱、新能源、创新科技等多个行业
4. **波动合理**：考虑季节性因素和行业特性的收入波动
5. **状态完整**：包含已支付、待支付、延迟支付等多种状态
